My experience with git and GitHub

Friday July 5, 2019

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. Having consistently implemented Git and GitHub as version control since the beginning of the University of Toronto bootcamp with success on our projects, it is with confidence that I say I am very comfortable and skilled at this element. It is convenient and reliable!
